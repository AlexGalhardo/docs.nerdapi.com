# What is NerdAPI.com?

We love games, we love programming, we grew up with games and we also love great developer experience APIs.

Our main goal is to make the best developer experience API for games information on the internet, with all the mainly statistics and info you need.

# How can I contribute to NerdAPI.com?
You can help us by being a subscription member, buy a game from our affiliate links, sharing the site with your friends and acquaintances, reporting bugs, and sugesting missing or wrong video game data.

# API Keys

To get your API Key:
- Create an account: https://nerdapi.com/register
- See and copy your API Key at your profile page: https://nerdapi.com/profile

# API Requests Limits

- NOOB Account (free forever): 25 API Requests/daily
- CASUAL Account: 1000 API Requests/daily
- PRO Account: 5000 API Requests/daily
