# What is NerdAPI.com?

We love games, we love programming, we grew up with games and we also love great developer experience APIs.

Our main goal is to make the best developer experience API for games information on the internet, with all the mainly statistics and info you need.

# How can I contribute to NerdAPI.com?
The best way to contribute is to use NerdAPI.com, searching and adding information about games. If you like the site and where we are heading please tell your friends about it as well.

You can also contribute with being a member of one of our subscriptions plans.

# Api Keys

To get access to a API KEY you need to have a subscription plan Casual or PRO.

To get your subscription, access: https://nerdapi.com/pricing
