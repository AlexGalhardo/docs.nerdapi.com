---
title: "GET GAME BY TITLE"
api: "GET https://api.nerdapi.com/games/title/:game_title"
description: "This endpoint returns array of games according to the searched title."
---

### Headers

<ParamField body="Authorization Bearer" type="string">
	Send in each request your API KEY in the request header
</ParamField>

<RequestExample>

```javascript Using Javascript
fetch(`https://api.nerdapi.com/games/title/godofwar`, {
		method: "GET",
		headers: { 
			"Content-Type": "application/json",
			"Authorization": "Bearer <YOUR_API_KEY>"
		}
 	})
 	.then(res => res.json())
 	.then(json => console.log(json));
```

```bash Using cURL
curl --location --request GET "https://api.nerdapi.com/games/title/godofwar" \
--header "Content-Type: application/json" \
--header "Authorization: Bearer <YOUR_API_KEY>" \
```

</RequestExample>
